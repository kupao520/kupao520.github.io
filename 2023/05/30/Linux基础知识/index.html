|阅读量:

Linux基础操作指令：

  • ```
    cd

    列出当前目录下的文件和文件夹

    • ls：列出当前目录下的所有文件和文件夹
    • ls -l：以详细列表的形式列出当前目录下的所有文件和文件夹

  • ```
    pwd

    创建一个新的目录

    • mkdir newdir：在当前目录下创建一个名为 newdir 的新目录

  • ```
    rm

    复制文件或目录

    • cp file.txt /home/user/Documents：将当前目录下名为 file.txt 的文件复制到 /home/user/Documents 目录下
    • cp -r olddir newdir：将当前目录下名为 olddir 的目录及其所有子目录和文件复制到当前目录下的名为 newdir 的新目录中

  • ```
    mv

    创建一个新文件或更新已有文件的时间戳

    • touch newfile.txt：在当前目录下创建一个名为 newfile.txt 的新文件
    • touch file.txt：更新当前目录下名为 file.txt 的文件的时间戳

  • ```
    cat

    在文件中查找指定的字符串

    • grep "hello" file.txt：在当前目录下名为 file.txt 的文件中查找包含字符串 “hello” 的行

  • ```
    chown

    显示当前正在运行的进程

  • ```
    top

    终止一个进程

    • kill 1234：终止进程号为 1234 的进程

  • ```
    ifconfig

    测试网络连接是否正常

    • ping www.baidu.com：测试与 www.baidu.com 的网络连接是否正常

  • ```
    ssh

    在计算机之间复制文件

    • scp file.txt user@192.168.1.100:/home/user/Documents：将当前目录下名为 file.txt 的文件复制到 IP 地址为 192.168.1.100 的计算机的 /home/user/Documents 目录下
  • wget

    从网络上下载文件

    • wget http://www.example.com/file.txt：从 http://www.example.com 网站下载名为 file.txt 的文件到当前目录下

以上就是Linux基本操作指令中的一部分，可以完成大多数情况下的业务需求
