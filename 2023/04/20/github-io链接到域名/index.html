发表于2023-04-20|更新于2023-04-20
|阅读量:

github.io链接到域名

1.进入自己的仓库

2.点击Add file ->> Creat new file

3.点击Name your file输入框并输入CNAME

4.在下面输入自己的网站域名

5.拉到最下面点击绿色按钮 Commit new file

6.回到主页点击CNAME文件，在点击这个点，当全部通过时绑定域名成功

7.在自己电脑上Win+R运行cmd命令窗口，ping一下自己的user.github.io ，得到一个IP地址

8.添加域名解析记录，此处以腾讯云为例，添加两个 A 记录，用 ping 所得 IP，一个主机记录为：“www”，另一个为“@”。

稍等几分钟，此后通过ikun.com 和www.ikun.com都能进行访问了。
